## .gitignore

.gitignore은 Git 버전 관리에서 제외할 파일 목록을 지정하는 파일이다.

git으로 프로젝트를 관리할 때, 관리가 불필요한 파일들을 제외 시킬 수 있다. 


> 이미 올라간 파일 삭제

이미 올라간 파일에 .gitignore 파일에 기록한다고 해서 버전 관리에서 제외하지 않기 때문에
수동으로 삭제 해주어야한다.
