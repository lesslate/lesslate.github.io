## Intro

> - JSX란

## JSX

`javascript`를 확장한 문법으로 자바스크립트 내부에 `HTML`을 작성할 수 있다.

`babel-loader`를 통해 자바스크립트로 변환된다.

## 예시

```js
function App()
{
  return (
    <div>
      <h1>Hello</h1>
    </div>
  );
}
```
